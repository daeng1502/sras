=== PERSYARATAN UTAMA SISTEM (WINDOWS) ===

1. RUNTIME & ENVIROMENT:
   - Node.js LTS (Versi 18.x / 20.x atau lebih baru)
     Unduh: https://nodejs.org/
   - npm (Otomatis terinstal bersama Node.js)

2. BROWSER & CHROME DRIVER:
   - Google Chrome (Versi stabil terbaru terinstal di Windows)
   - Chromium (Otomatis diunduh secara internal oleh library Puppeteer saat instalasi)

3. DEPENDENSI NPM (TERTULIS DI PACKAGE.JSON):
   - whatsapp-web.js (v1.25.0)  -> Pustaka utama otomasi WhatsApp Web
   - qrcode-terminal (v0.12.0)  -> Penampil QR Code di terminal CMD
   - dotenv (v16.4.5)           -> Pengatur konfigurasi lingkungan (.env)
   - jest (v29.7.0)             -> Kerangka kerja pengujian offline (Development)

4. PERALATAN NOTIFIKASI ALARM HP (OPSIONAL):
   - Aplikasi 'ntfy' di Android (Unduh gratis dari Google Play Store)
   - Topik langganan unik ntfy (contoh: sras_alarm)

=== LANGKAH INSTALASI DAN PENJALANAN ===

1. Pastikan Node.js sudah terinstal dengan mengetik di CMD/PowerShell:
   node -v
   npm -v

2. Jalankan instalasi seluruh pustaka dependensi:
   npm install

3. Konfigurasi file '.env' di folder utama proyek.

4. Jalankan bot:
   npm start
